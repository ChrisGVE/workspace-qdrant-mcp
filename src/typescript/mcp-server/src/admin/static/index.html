Global index exclusions

          applied to every project
        

        

          Patterns are applied on top of each project's own .gitignore /
          .wqmignore. Uses standard gitignore syntax:
          node_modules/, *.zip, !important.log.
          Changes take effect on the next ignore-file reconciliation (triggered automatically
          when you save any project's .gitignore or .wqmignore,
          or at the next daemon restart).

          injected into every agent session
        

        

          Global rules apply to every project; project rules
          apply only to the selected project. Both are merged (global first, by priority) at
          session start and injected into the agent's system prompt.
