## Validation

Run the validation script to verify file classification after ingestion:

```bash
# Register and validate (full workflow)
./validate.sh --register --wait --verbose

# Validate only (if already registered)
./validate.sh --verbose
```

## Regenerating Binary Files

Binary files (PDF, DOCX, PPTX, ODT, ODP, EPUB, RTF) are generated by a Python script:

```bash
python3 generate_binaries.py
```

This script uses only Python standard library (zipfile, struct, zlib) with no external dependencies.
